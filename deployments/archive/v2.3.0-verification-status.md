## Verification Commands Used

### For Implementation Contracts
```bash
# EscrowDst on Optimism
forge verify-contract \
  --chain-id 10 \
  --num-of-optimizations 200 \
  --constructor-args $(cast abi-encode "constructor(uint32,address)" 604800 0x8287CD2aC7E227D9D927F998EB600a0683a832A1) \
  --etherscan-api-key $OPTIMISM_ETHERSCAN_API_KEY \
  --compiler-version v0.8.23+commit.f704f362 \
  0x32e98F40D1D4643b251D8Ee99fd95918A3A8b306 \
  contracts/EscrowDst.sol:EscrowDst
```
